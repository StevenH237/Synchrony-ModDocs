Switcheroo has the following entities of its own:

# `Switcheroo_NoneItem`
This is a placeholder entity that represents the lack of an item in certain settings. It only contains the following component:

```lua
friendlyName = {
  name = "(None)"
}
```

# `Switcheroo_RandomChannel`
This is an entity that denotes the random channel for a given player's item randomization. It only contains the following component:

```lua
random = {}
```
